## Concise (pick 3–5)

- Built a **production-ready full-stack e-commerce platform** (Node.js, Express,
  React, TypeScript, PostgreSQL, Drizzle ORM) with a customer storefront and a
  data-rich admin dashboard, deployed on **AWS Elastic Beanstalk** with **RDS
  PostgreSQL**.

- Designed and shipped a **weekly subscription system** (Saturday/Sunday
  deliveries) with full lifecycle management — pause, resume, skip, cancel,
  reactivate, change-plan — and automated weekly order generation.

- Implemented a **rules-based discounting and referral engine**: first-order
  discounts, a referral program (10% off for new customers, 5% reward per
  successful referral) with **abuse protection and a 30%-per-order reward cap**,
  all re-priced server-side to prevent client tampering.

- Integrated the **PhonePe Payment Gateway (v2 OAuth)** end to end — server-side
  payment initiation, status verification, signed webhooks, and refunds — with
  transactional records for reconciliation and a simulation mode for testing
  without live credentials.

- Delivered **Power BI analytics** via 10 read-only PostgreSQL reporting views
  (sales, payments, subscriptions, referrals, inventory, customer value) exposed
  through a dedicated least-privilege BI database role.

## Longer form (single paragraph)

Developed FarmFreshFarmer, a production-ready full-stack e-commerce application
(Node.js/Express/TypeScript backend, React/Tailwind frontend, PostgreSQL +
Drizzle ORM) deployed to AWS Elastic Beanstalk with RDS PostgreSQL. Built a
complete admin dashboard, a weekly (Sat/Sun) subscription platform with full
lifecycle controls, a server-side discount and referral engine with abuse
protection and reward caps, and an end-to-end PhonePe (v2 OAuth) payment
integration with webhook verification and refunds. Exposed the operational data
to Power BI through 10 read-only reporting views behind a least-privilege role,
and automated schema migrations and view provisioning on every deploy.
